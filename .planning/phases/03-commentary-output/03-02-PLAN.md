Create `src/commentary/tts_engine.py` with class **TTSEngine**:

Constructor: `__init__(self, api_key: str, voice_id: str, event_bus: EventBus | None = None)`
- Store api_key, voice_id, event_bus reference.
- Create `AsyncCartesia(api_key=api_key)` client.
- Initialize `_ws = None` (WebSocket connection, created on connect).
- Initialize PyAudio instance and output stream as None (created on connect).
- Set `_sample_rate = 22050`.
- Set `_connected = False` flag.

Method: `async def connect(self) -> None`
- Open Cartesia WebSocket: `self._ws = await self._client.tts.websocket()`
- Open PyAudio output stream: `self._pyaudio.open(format=pyaudio.paFloat32, channels=1, rate=self._sample_rate, output=True)`.
- Set `_connected = True`.
- Log connection success.

Method: `async def speak(self, sentence: str, context_id: str, emotion: str = "sarcastic", is_continuation: bool = False) -> None`
- If not connected, log warning and return (graceful degradation).
- Publish TTSSpeaking event on the event bus (so audio capture can mute).
- Call `self._ws.send()` with:
  - `model_id="sonic-3"`
  - `transcript=sentence`
  - `voice={"id": self._voice_id}`
  - `stream=True`
  - `context_id=context_id`
  - `continue_=is_continuation`
  - `output_format={"container": "raw", "encoding": "pcm_f32le", "sample_rate": self._sample_rate}`
  - `generation_config={"speed": 1.1, "emotion": emotion}`
- Iterate over output chunks: `async for out in output: if out.audio: await asyncio.to_thread(self._stream.write, out.audio)`
- Publish TTSFinished event on the event bus (so audio capture can unmute).
- Wrap the entire method body in try/except. On ANY exception, log the error and publish TTSFinished anyway (never leave capture muted on TTS failure).

Method: `async def speak_commentary(self, sentences: list[str], emotion_map: dict[int, str] | None = None) -> None`
- Generate a unique context_id using `str(uuid.uuid4())[:8]`.
- For each sentence with index i:
  - Determine emotion from emotion_map (default "sarcastic").
  - Call `await self.speak(sentence, context_id, emotion, is_continuation=(i > 0))`.

Method: `async def close(self) -> None`
- Close WebSocket if open.
- Close PyAudio stream and terminate PyAudio instance.
- Set `_connected = False`.

Import TTSSpeaking, TTSFinished from src.commentary.models. Import EventBus from src.capture.event_bus.

Create `src/commentary/display_server.py` with class **DisplayServer**:

**ConnectionManager** (inner class or module-level):
- `active: list[WebSocket] = []`
- `async def connect(self, ws: WebSocket)` — accept and append.
- `def disconnect(self, ws: WebSocket)` — remove from list.
- `async def broadcast(self, message: dict)` — send_json to all active, catch exceptions for disconnected clients.

**DisplayServer** class:
Constructor: `__init__(self, host: str = "0.0.0.0", port: int = 8080)`
- Create FastAPI app with title "Arbiter Display".
- Create ConnectionManager instance.
- Set up Jinja2Templates pointing to `src/commentary/templates`.
- Register routes on the app:
  - `GET /` — Serve the display.html template via Jinja2.
  - `WebSocket /ws/display` — Accept WebSocket connections, add to manager, keep alive with receive loop, disconnect on close.
- Store host, port, uvicorn server/config as None (created on start).

Method: `async def start(self) -> None`
- Create `uvicorn.Config(app=self._app, host=self._host, port=self._port, log_level="warning")`.
- Create `uvicorn.Server(config)`.
- Run `server.serve()` as an asyncio task (not blocking). Store the task.
- Log: "Display server started on http://{host}:{port}"

Method: `async def push_commentary(self, text: str, team_name: str) -> None`
- Broadcast `{"type": "commentary", "text": text, "team_name": team_name}` via ConnectionManager.

Method: `async def push_question(self, text: str, team_name: str) -> None`
- Broadcast `{"type": "question", "text": text, "team_name": team_name}` via ConnectionManager.

Method: `async def clear(self) -> None`
- Broadcast `{"type": "clear"}` via ConnectionManager.

Method: `async def stop(self) -> None`
- Signal uvicorn server shutdown. Cancel server task if running.
