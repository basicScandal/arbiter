Wire all defense components into a DefensePipeline orchestrator and integrate it with the existing CapturePipeline. This is the integration plan that makes the defense layer operational.

Purpose: Individual defense components (OCR, detector, sanitizer, roaster, logger) work in isolation but need event bus orchestration to intercept capture events and process them through the defense chain. The DefensePipeline subscribes to capture events, runs detection, fires roasts, logs attempts, and on demo stop produces a clean SanitizedOutput for downstream Phase 3/4 consumers. The CapturePipeline is updated to wire the defense layer alongside capture.

Output: A working defense pipeline that intercepts live capture data, detects injection attempts, generates roasts, and produces sanitized observations.

    **src/defense/pipeline.py**: Event-driven defense orchestrator.

    - `class DefensePipeline`:
      - `__init__(self, api_key: str)`: Create all defense components:
        - `self._ocr = OCRScanner()`
        - `self._detector = InjectionDetector()`
        - `self._sanitizer = ObservationSanitizer(self._detector)`
        - `self._roaster = RoastGenerator(api_key=api_key)`
        - `self._logger = InjectionLogger()`
        - `self._event_bus: EventBus | None = None`
        - `self._current_team: str = ""`
        - `self._roasts: list[str] = []` -- accumulated roasts for current demo
        - `self._pending_roast_tasks: list[asyncio.Task] = []` -- track detached roast tasks

      - `async def setup(self, event_bus: EventBus) -> None`: Subscribe to capture events:
        - `event_bus.subscribe("key_frame_detected", self._on_key_frame)`
        - `event_bus.subscribe("transcript_received", self._on_transcript)`
        - `event_bus.subscribe("demo_started", self._on_demo_started)`
        - `event_bus.subscribe("demo_stopped", self._on_demo_stopped)`
        - Store event_bus reference for publishing.
        - Log "Defense pipeline armed" at INFO level.

      - `async def _on_demo_started(self, event: DemoStarted) -> None`:
        - Set `self._current_team = event.team_name`.
        - Clear `self._roasts` and `self._logger` for fresh demo.
        - Log "Defense pipeline active for team: {team}".

      - `async def _on_key_frame(self, event: KeyFrameDetected) -> None`:
        - Run OCR via asyncio.to_thread: `ocr_text = await asyncio.to_thread(self._ocr.extract_text, event.frame.jpeg_data)`.
        - If ocr_text is empty, return early (no text to scan).
        - Scan for visual injection: `result = self._detector.scan_visual(ocr_text)`.
        - If result.is_injection:
          - Create InjectionAttempt with timestamp=event.timestamp, injection_type="visual", content=result.matched_text, pattern=",".join(result.matched_patterns), confidence=result.confidence, team_name=self._current_team.
          - Log the attempt via self._logger.log(attempt).
          - Publish InjectionDetected event to event bus.
          - If result.confidence == "high": fire roast as detached task via `asyncio.create_task(self._generate_roast(attempt))`. Append task to _pending_roast_tasks.

      - `async def _on_transcript(self, event: TranscriptReceived) -> None`:
        - Scan for verbal injection: `result = self._detector.scan_verbal(event.segment.text)`.
        - If result.is_injection:
          - Create InjectionAttempt with timestamp=event.timestamp, injection_type="verbal", content=result.matched_text or event.segment.text[:200], pattern=",".join(result.matched_patterns), confidence=result.confidence, team_name=self._current_team.
          - Log the attempt via self._logger.log(attempt).
          - Publish InjectionDetected event.
          - If result.confidence == "high": fire roast as detached task.

      - `async def _generate_roast(self, attempt: InjectionAttempt) -> None`:
        - Generate roast: `roast = await self._roaster.generate(attempt)`.
        - Append to self._roasts.
        - Publish RoastGenerated event to event bus.
        - Log the roast at INFO level: "ROAST: {roast}".

      - `async def _on_demo_stopped(self, event: DemoStopped) -> None`:
        - Wait for any pending roast tasks to complete (with short timeout): `await asyncio.gather(*self._pending_roast_tasks, return_exceptions=True)` wrapped in asyncio.wait_for with 5-second timeout.
        - Clear _pending_roast_tasks.
        - Get Gemini observations from the capture pipeline. Access via the event -- the CapturePipeline stores observations in DemoSession. The defense pipeline needs the observations list. Since the observations are stored in the DemoSession by the CapturePipeline._on_demo_stopped handler, and that handler runs concurrently with this one, we need a different approach: the DefensePipeline should receive observations via a method call from the CapturePipeline rather than event bus timing.
        - **Approach: `set_observations(self, observations: list[str]) -> None` method** that the CapturePipeline calls before publishing demo_stopped, or the DefensePipeline can access gemini.get_observations() directly. Better: pass the GeminiSession reference at setup time.
        - **Revised approach:** `__init__` also accepts an optional `gemini_session` parameter (the GeminiSession from Phase 1). In _on_demo_stopped, call `self._gemini.get_observations()` to get raw observations.
        - Scan each observation with `self._detector.scan_observation(obs)`. Log any additional injection attempts found.
        - Call `self._sanitizer.create_sanitized_output(team_name=event.team_name, observations=raw_observations, transcripts=self._get_transcripts(), attempts=self._logger.get_attempts(), duration=event.duration, roasts=list(self._roasts))`.
        - Publish ObservationVerified event with the SanitizedOutput.
        - Log summary: "Defense summary: {N} injection attempts, {M} observations sanitized, {R} roasts generated".

      **Wait -- transcripts.** The defense pipeline sees transcripts one at a time via _on_transcript events but needs the full list for sanitization. Solution: accumulate transcript text in `self._transcripts: list[str] = []` during demo. Clear on demo_started.

      So update __init__ to also have: `self._transcripts: list[str] = []`
      Update _on_demo_started to also clear self._transcripts.
      Update _on_transcript to append event.segment.text to self._transcripts (regardless of injection status -- the sanitizer handles filtering).

      Final __init__ signature: `__init__(self, api_key: str, gemini_session: GeminiSession | None = None)`.
