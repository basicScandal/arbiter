    2. **src/defense/models.py**: Pydantic models for the defense pipeline:

       - `InjectionPattern(BaseModel)`: Defines a single regex-based injection detection pattern.
         - `name: str` -- pattern identifier (e.g., "ignore_previous")
         - `pattern: str` -- regex pattern string
         - `severity: str` -- "high", "medium", or "low"
         - `category: str` -- one of: "instruction_override", "scoring", "role_manipulation", "extraction", "context_escape"

       - `DetectionResult(BaseModel)`: Result from scanning text against injection patterns.
         - `is_injection: bool` -- whether any pattern matched
         - `matched_patterns: list[str]` -- names of all patterns that matched (empty list default)
         - `matched_text: str` -- the specific text that triggered the match (empty string default)
         - `confidence: str` -- "high" (2+ high-severity matches), "medium" (1 high or 2+ medium), "low" (1 medium/low match). Default "low"
         - `source: str` -- "visual", "verbal", or "observation" indicating scan origin

       - `InjectionAttempt(BaseModel)`: A confirmed injection attempt for logging and roasting.
         - `timestamp: float`
         - `injection_type: str` -- "visual" or "verbal" or "observation"
         - `content: str` -- the matched text that triggered detection
         - `pattern: str` -- comma-joined names of matched patterns
         - `confidence: str` -- from DetectionResult
         - `team_name: str` -- default empty string

       - `SanitizedOutput(BaseModel)`: Clean output bundle for downstream consumers (Phase 3/4).
         - `team_name: str`
         - `observations: list[str]` -- clean Gemini observations (tainted ones removed)
         - `transcripts: list[str]` -- clean transcript text segments
         - `injection_attempts: list[InjectionAttempt]` -- all detected attempts
         - `demo_duration: float`
         - `roasts: list[str]` -- generated roast responses (default empty list)

       - Defense event types extending `CaptureEvent` from `src.capture.models`:
         - `InjectionDetected(CaptureEvent)`: event_type="injection_detected", attempt: InjectionAttempt
         - `RoastGenerated(CaptureEvent)`: event_type="roast_generated", roast: str, attempt: InjectionAttempt
         - `ObservationVerified(CaptureEvent)`: event_type="observation_verified", output: SanitizedOutput

    Import CaptureEvent from src.capture.models. Use `from __future__ import annotations` at top.

    2. **src/defense/ocr_scanner.py**: OCR text extraction from key frame images.

       - `class OCRScanner`:
         - `__init__(self)`: Check Tesseract availability at construction time via `pytesseract.get_tesseract_version()`. If not available, log a WARNING and set `self._available = False`. Do NOT crash -- OCR is a defense layer, not a critical path.
         - `@property available(self) -> bool`: Returns self._available.
         - `def extract_text(self, jpeg_data: bytes, timeout: int = 5) -> str`: Synchronous method (will be called via asyncio.to_thread by pipeline).
           - If not self._available, return "".
           - Decode JPEG bytes to numpy array via `np.frombuffer(jpeg_data, np.uint8)` then `cv2.imdecode(nparr, cv2.IMREAD_COLOR)`.
           - If decode fails (None), return "".
           - Preprocess: convert to grayscale (`cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)`), apply adaptive thresholding (`cv2.adaptiveThreshold(gray, 255, cv2.ADAPTIVE_THRESH_GAUSSIAN_C, cv2.THRESH_BINARY, 11, 2)`).
           - Extract text: `pytesseract.image_to_string(thresh, config="--psm 3", timeout=timeout)`.
           - Wrap in try/except RuntimeError (timeout) -- return "" on timeout.
           - Return text.strip().

       Import cv2, numpy as np, pytesseract, logging. Use `from __future__ import annotations`.

    3. **src/defense/injection_detector.py**: Regex-based injection pattern detection.

       - Define `INJECTION_PATTERNS: list[InjectionPattern]` at module level with these categories:

         **instruction_override** (high severity):
         - `ignore_previous`: `r"(?i)\b(ignore|disregard|forget)\b.{0,30}\b(previous|above|prior|all)\b.{0,30}\b(instructions?|prompts?|rules?)\b"`
         - `new_instructions`: `r"(?i)\b(new|updated|real|actual|true)\b.{0,20}\b(instructions?|directives?|orders?|rules?)\b"`

         **scoring** (high severity):
         - `score_manipulation`: `r"(?i)\b(give|assign|score|rate|award)\b.{0,30}\b(10|perfect|full|maximum|highest)\b"`
         - `score_override`: `r"(?i)\b(score|rating|points?)\b.{0,20}\b(must be|should be|is now|set to)\b.{0,10}\b\d"`

         **role_manipulation** (medium severity):
         - `role_override`: `r"(?i)\b(you are|act as|pretend|behave as|your role is)\b.{0,30}\b(not a judge|helpful assistant|friendly|compliant)\b"`
         - `identity_reset`: `r"(?i)\b(you are now|from now on|henceforth|starting now)\b"`

         **extraction** (medium severity):
         - `prompt_extraction`: `r"(?i)\b(system prompt|initial prompt|reveal|show|display|repeat)\b.{0,30}\b(instructions?|prompt|rules?|configuration)\b"`
         - `output_rules`: `r"(?i)\b(print|output|display|echo|return)\b.{0,20}\b(above|system|hidden|secret|internal)\b"`

         **context_escape** (high severity):
         - `delimiter_escape`: `r"(?i)(```|</?system>|</?user>|</?assistant>|\[INST\]|\[/INST\]|<<SYS>>|<</SYS>>)"`
         - `xml_injection`: `r"(?i)(<\/?(?:tool|function|command|execute|admin|root)>)"`

       - `class InjectionDetector`:
         - `__init__(self, patterns: list[InjectionPattern] | None = None)`: Use INJECTION_PATTERNS by default.
         - `def scan(self, text: str, source: str) -> DetectionResult`: Core scanning method.
           - If text is empty or whitespace, return DetectionResult(is_injection=False, source=source).
           - Run each pattern's regex against text via `re.search(pattern.pattern, text)`.
           - Collect all matches: pattern names, matched text (from first match).
           - Compute confidence:
             - Count high-severity matches and medium-severity matches.
             - If high_count >= 2: confidence = "high"
             - If high_count == 1 or medium_count >= 2: confidence = "medium"
             - Else: confidence = "low"
           - Return DetectionResult with is_injection=(len(matches) > 0), matched_patterns, matched_text (from first match group or matched string), confidence, source.
         - `def scan_visual(self, text: str) -> DetectionResult`: Calls `self.scan(text, source="visual")`.
         - `def scan_verbal(self, text: str) -> DetectionResult`: Calls `self.scan(text, source="verbal")`.
         - `def scan_observation(self, text: str) -> DetectionResult`: Calls `self.scan(text, source="observation")`.

       Import re, logging. Import InjectionPattern, DetectionResult from src.defense.models. Use `from __future__ import annotations`.

       **Key design note on false positives (per research Pitfall 2):** This is a security hackathon -- presenters will legitimately discuss "prompt injection", "system prompts", etc. The confidence scoring (requiring multiple pattern matches for high confidence) prevents over-triggering roasts. Only HIGH confidence triggers roasts (handled in Plan 02-03 pipeline). All detections are logged regardless of confidence.
