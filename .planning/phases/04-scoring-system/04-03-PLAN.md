  <action>
Create `src/scoring/pipeline.py` implementing `ScoringPipeline`:

```python
class ScoringPipeline:
    """Orchestrates scoring: engine call, persistence, and theatrical display reveal.

    Subscribes to observation_verified to score demos and commentary_delivered
    to trigger the theatrical score reveal AFTER commentary finishes.
    This ensures scores appear after Arbiter's verbal commentary, creating
    natural dramatic timing (Pitfall 5 avoidance).
    """
```

Constructor:
- `__init__(self, api_key: str, display: DisplayServer, scores_dir: str = "data/scores")`
- Creates `ScoringEngine(api_key=api_key)` -- this is the isolated scoring engine
- Creates `ScoreStore(scores_dir=scores_dir)`
- Stores display reference (the DisplayServer from CommentaryPipeline)
- `self._pending_scorecards: dict[str, DemoScorecard] = {}` -- maps team_name to scorecard awaiting reveal
- `self._pending_tracks: dict[str, str] = {}` -- maps team_name to track, set by operator
- `self._event_bus: EventBus | None = None`

Methods:

`async def setup(self, event_bus: EventBus) -> None`:
- Store event_bus reference
- Subscribe to "observation_verified" -> `self._on_observation_verified`
- Subscribe to "commentary_delivered" -> `self._on_commentary_delivered`
- Log "Scoring pipeline armed"

`def set_track(self, team_name: str, track: str) -> None`:
- Stores track assignment: `self._pending_tracks[team_name] = track`
- Called by operator CLI when starting a demo with track argument

`async def _on_observation_verified(self, event: ObservationVerified) -> None`:
- Extract team_name from event.output.team_name
- Look up track from `self._pending_tracks.get(team_name, "ROGUE::AGENT")` (default to ROGUE::AGENT if no track specified -- it's the "novel/uncategorized" track)
- Call `self._engine.score(event.output, track)` wrapped in try/except
- On success: store scorecard in `self._pending_scorecards[team_name]`, persist via `self._store.save(scorecard)`, log score
- On error: log exception as warning, do NOT crash pipeline
- Publish ScoringComplete event with the scorecard

`async def _on_commentary_delivered(self, event: CommentaryDelivered) -> None`:
- Look up scorecard from `self._pending_scorecards.pop(event.team_name, None)`
- If no scorecard ready (scoring still in progress or failed), log and return
- Launch score reveal as detached task: `asyncio.create_task(self._reveal_score(scorecard))`
- This is critical: the reveal must NOT block the event bus callback (Pitfall 3)

`async def _reveal_score(self, scorecard: DemoScorecard) -> None`:
- Try/except the entire reveal (never crash on display errors)
- Phase 1: `await self._display.push_score_intro(scorecard.team_name)` then `await asyncio.sleep(2.0)`
- Phase 2: For each criterion in scorecard.criteria:
  - `await self._display.push_criterion_reveal(criterion.name, criterion.score, criterion.weight, criterion.justification)`
  - `await asyncio.sleep(1.5)`
- Phase 3: If track_bonus exists, reveal it too with `push_criterion_reveal` and 1.5s pause
- Phase 4: `await asyncio.sleep(1.0)` then `await self._display.push_total_score(scorecard.team_name, scorecard.total_score, scorecard.track)`
- Publish ScoreRevealed event
- Log "Score reveal complete for team: {team_name}"
