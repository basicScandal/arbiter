Create `src/memory/store.py` following the EXACT pattern of `src/scoring/store.py`:

**MemoryStore class:**

`__init__(self, observations_dir: str = "data/observations")`:
- Store as `self._dir = Path(observations_dir)`
- `self._dir.mkdir(parents=True, exist_ok=True)`

`_sanitize_team_name(team_name: str) -> str` (static method):
- Copy the EXACT implementation from `ScoreStore._sanitize_team_name()` -- replace spaces with underscores, strip non-alphanumeric except underscores/hyphens, lowercase. Per research: "extract to shared utility or import from scoring" -- but to avoid touching Phase 4 files, duplicate the 3-line static method.

`async save(self, memory: DemoMemory) -> Path`:
- Sanitize team name
- Path is `self._dir / f"{sanitized}.json"`
- Serialize with `json.dumps(memory.model_dump(), indent=2, default=str)`
- Write via `await asyncio.to_thread(path.write_text, data)`
- Log at INFO level
- Return the path

`async load(self, team_name: str) -> DemoMemory | None`:
- Sanitize, check path exists, read via asyncio.to_thread, `DemoMemory.model_validate_json(raw)`
- Return None if not found

`async load_all(self) -> list[DemoMemory]`:
- Iterate sorted `self._dir.glob("*.json")`
- Each: try read + model_validate_json, except log warning and skip
- Return list

Imports: asyncio, json, logging, re, Path. From models import DemoMemory.
