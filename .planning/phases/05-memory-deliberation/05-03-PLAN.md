**1. Create `src/memory/pipeline.py`** following the pattern of `src/scoring/pipeline.py`:

**DeliberationPipeline class:**

`__init__(self, api_key: str, display: DisplayServer, scores_dir: str = "data/scores", observations_dir: str = "data/observations", deliberation_dir: str = "data/deliberation")`:
- `self._memory_store = MemoryStore(observations_dir=observations_dir)`
- `self._score_store = ScoreStore(scores_dir=scores_dir)`
- `self._engine = DeliberationEngine(api_key=api_key)`
- `self._display = display`
- `self._deliberation_dir = Path(deliberation_dir)` -- mkdir parents=True, exist_ok=True
- `self._pending_tracks: dict[str, str] = {}` -- track lookup for memory storage
- `self._event_bus: EventBus | None = None`

`async setup(self, event_bus: EventBus)`:
- Store event_bus reference
- Subscribe to `observation_verified` -> `_on_observation_verified` (save memory)
- Subscribe to `deliberation_requested` -> `_on_deliberation_requested` (run deliberation)
- Log "Deliberation pipeline armed"

`set_track(self, team_name: str, track: str)`:
- Store track for team (same pattern as ScoringPipeline.set_track)

`async _on_observation_verified(self, event: ObservationVerified)`:
- Extract from event.output: team_name, observations, transcripts, injection_attempts (len), demo_duration
- Look up track from `self._pending_tracks.get(team_name, "ROGUE::AGENT")`
- Create DemoMemory with stored_at=time.time()
- Save via memory_store.save()
- Log at INFO: "Memory saved for team {team_name} ({N} observations)"
- Wrap in try/except -- log warning on failure, never crash pipeline

`async _on_deliberation_requested(self, event: DeliberationRequested)`:
- Load all memories and scorecards
- Guard: if len(memories) == 0, log warning "No demos recorded", return
- Guard: if len(memories) != len(scorecards), log warning with counts: "Observation/score mismatch: {M} observations, {S} scorecards -- proceeding anyway"
- Call engine.deliberate(memories, scorecards) in try/except
- On success: save result to `data/deliberation/result.json` via asyncio.to_thread
- Push to display via detached asyncio.create_task (same pattern as ScoringPipeline._reveal_score)
- Publish DeliberationComplete event
- On failure: log error with exc_info=True

`async _push_deliberation_display(self, result: DeliberationResult)`:
- Push each ranking sequentially with 2-second spacing between teams:
  ```
  for ranking in result.rankings:
      await self._display.push_deliberation_ranking(
          rank=ranking.rank,
          team_name=ranking.team_name,
          total_score=ranking.total_score,
          track=ranking.track,
          reasoning=ranking.reasoning,
      )
      await asyncio.sleep(2.0)
  ```
- After all rankings, push the overall narrative:
  ```
  await self._display.push_deliberation_narrative(result.overall_narrative)
  ```
- Wrap in try/except with warning log (display errors must not crash)
