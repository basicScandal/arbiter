**2. `src/rehearsal/replay_provider.py`** — An `LLMProvider` that returns canned responses:
- `class ReplayProvider(LLMProvider)` implementing `generate()` and `name` property
- Constructor accepts optional `responses: dict[str, str]` for custom canned responses
- `name` property returns `"replay"`
- `generate()` inspects prompt/system_prompt keywords to determine which canned response to return:
  - If system_prompt contains "scoring" (case-insensitive) → return canned scoring JSON response. The scoring response MUST be valid JSON matching `ScoringEngine._parse_and_validate` expectations: a JSON object with a `"criteria"` array where each item has `"name"` (str), `"score"` (float 0-10), and `"justification"` (str). Use realistic scores (not all 5.0): e.g., Technical Execution: 8.5, Innovation: 7.0, Demo Quality: 6.5. Include a `"track_bonus"` object if the prompt mentions a specific track.
  - If system_prompt contains "arbiter" or "persona" (case-insensitive) → return canned commentary text. Make it sound like the Simon Cowell Arbiter persona (~2-3 sentences, opinionated and entertaining).
  - If system_prompt contains "deliberat" (case-insensitive) → return canned deliberation JSON
  - If prompt contains "question" or "Q&A" → return a canned Q&A question
  - Default: return a generic placeholder response
- Store `_call_count: int` incremented on each call for testing/debugging
- All responses are synchronous string returns (no actual API calls, no sleeps)

**3. `src/rehearsal/synthetic_capture.py`** — Publishes mock demo events:
- `class SyntheticCapture` with `__init__(self, event_bus: EventBus)`
- `async def run_demo(self, team_name: str = "RehearsalTeam", track: str = "ROGUE::AGENT", duration: float = 180.0) -> None`:
  - Publishes `DemoStarted(team_name=team_name)`
  - Brief `await asyncio.sleep(0.5)` for event propagation
  - Publishes 3 `KeyFrameDetected` events with synthetic `FrameData` (minimal valid data: 1x1 pixel JPEG bytes, width=1, height=1, is_key_frame=True)
  - Publishes 3 `TranscriptReceived` events with synthetic `TranscriptSegment` data (realistic rehearsal transcript text describing a mock demo)
  - Brief sleep between events (0.2s)
  - Publishes `DemoStopped(team_name=team_name, duration=duration)`
- Helper `_make_synthetic_frame() -> FrameData` that creates a minimal valid FrameData
- Module-level `_SYNTHETIC_TRANSCRIPTS: list[TranscriptSegment]` with 3 realistic rehearsal transcript segments
